# Minecraft Oracle - Seed Scanner Mod

Client-side mod for locating Diamond veins and Ancient Debris using the World Seed.

## 🚀 Features
- **Seed Synchronization:** User inputs seed to initialize scanning.
- **Overworld Mode:** Locates nearest Diamond veins.
- **Nether Mode:** Locates nearest Ancient Debris.
- **High-Frequency Vibe:** Developed with real-time acceleration.
